# Pokémon Explorer

Aplikacja umożliwia:
- przeglądanie listy Pokémonów
- wyszukiwanie Pokémonów po nazwie
- filtrowanie po typach
- dynamiczne ładowanie kolejnych Pokémonów

## Technologie
- React
- TypeScript
- Vite
- Tailwind CSS
- PokéAPI

## Do uruchomienia projektu potrzebne są:
- Node.js
- npm

## Uruchomienie projektu

```bash
git clone 
npm install
npm run dev
```
